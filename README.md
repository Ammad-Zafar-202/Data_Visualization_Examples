# Data_Visualization_Examples
Collection of work examples of Data Visualization using Python

Providing a basic introduction to Data Visualization for personal, work, or academic use, for free use.

Implementing Methods and Libraries such as:
- Plotting with Matplotlib
- Creating Line Plots
- Creating Area Plots
- Creating Histograms
- Creating Bar Charts
- Creating Pie Charts
- Creating Box Plots
- Creating Scatter Plots
- Creating Waffle Charts
- Creating Word Clouds
- Creating Box and Whisker Plots
- Creating Correlation Matrix
- Using Seaborn and Regression Plots
- Using Folium
- Mapping with Markers
- Creating Choropleth Maps
